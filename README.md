
# AffluenceBusiness

Office landing page for AffluenceBusiness.

## Tech Stack

**Stacks:** dJango, Docker


## Usage/Examples

First build docker image
```bash
docker-compose build
```

Start the container
```bash
docker-compose up -d
```
